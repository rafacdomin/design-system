- O que observar: estrutura de tokens (tiers: global → alias → component),

## Radix Themes

- Docs: https://radix-ui.com/themes
- O que observar: escala de tokens, sistema de cores com steps 1-12

## Base UI (MUI)

- Docs: https://base-ui.com
- O que observar: headless components, hooks de acessibilidade
