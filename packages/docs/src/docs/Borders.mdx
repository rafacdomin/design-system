# Bordas e Arredondamento

Os tokens de borda controlam o arredondamento de cantos (`border-radius`) e a espessura de bordas (`border-width`) de componentes. O design system adota uma estética angular e moderna com raios de borda discretos para construir uma interface geométrica e premium.

---

## 📐 Raio de Borda (Border Radius)

Usado para suavizar os cantos de botões, inputs, cards e contêineres.

# Borders and Radius

Border tokens control component corner rounding (`border-radius`) and border width (`border-width`). The design system adopts an angular and modern aesthetic with discrete border radii to build a geometric and premium interface.

---

## 📐 Border Radius

Used to soften the corners of buttons, inputs, cards, and containers.
