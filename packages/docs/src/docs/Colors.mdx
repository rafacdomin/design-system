# Cores

A paleta de cores do design system é focada em uma escala monocromática premium de alto contraste, utilizando tons cinza neutro para suavizar e dar profundidade à interface, além de manter a acessibilidade visual e conformidade com WCAG 2.1 AA.

Os valores de cor mudam dinamicamente dependendo do tema ativo (`light` ou `dark`). A documentação abaixo se adapta em tempo real quando você altera o tema no menu superior do Storybook.

---

## 🖤 Escala Neutra (Neutral Scale)

Esta escala é usada para fundos, containers, divisores, bordas e níveis hierárquicos de texto.

  </Language>
  <Language locale="en-US">

# Colors

The design system's color palette is focused on a premium high-contrast monochromatic scale, using neutral gray tones to soften and add depth to the interface, while maintaining visual accessibility and compliance with WCAG 2.1 AA.

Color values change dynamically depending on the active theme (`light` or `dark`). The documentation below adapts in real time when you change the theme in the top menu of Storybook.

---

## 🖤 Neutral Scale

This scale is used for backgrounds, containers, dividers, borders, and hierarchical text levels.
