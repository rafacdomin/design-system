# Sombras e Elevação

O design system utiliza sombras discretas baseadas em tons pretos translúcidos para gerar profundidade visual, separação de camadas e sensação de elevação física na interface.

---

## 🔲 Escala de Sombras (Shadow Scale)

As sombras são aplicadas em cards, modais, dropdowns e menus flutuantes para destacar elementos sobrepostos ao conteúdo principal.

# Shadows and Elevation

The design system uses discrete shadows based on translucent black tones to generate visual depth, layer separation, and a sense of physical elevation in the interface.

---

## 🔲 Shadow Scale

Shadows are applied to cards, modals, dropdowns, and floating menus to highlight elements overlaying the main content.
